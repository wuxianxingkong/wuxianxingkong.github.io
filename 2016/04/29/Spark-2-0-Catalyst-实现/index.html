Spark 2.0 Catalyst 实现

Posted on 2016-04-29   |   In Spark SQL   |     |  

针对于Catalyst中的Parser，与1.6.1区别

1.6.1版本中，有三个Parser，分析见之前的Spark SQL Catalyst分析

  • DDLParser
  • SparkSQLParser
  • SqlParser

2.0.0版本中，只保留了一个Parser。

Spark1.6以及之前版本的实现方式：

  • Scala解析器库、基于Hive的解析器
  • Spark2.0版本：使用ANTLR 4实现，参考博客链接：Antlr v4入门教程和实例
